Projects

Earthquake Data Pipeline

A fully automated data pipeline for processing and analyzing global earthquake events using modern data engineering tools. This project demonstrates the design and orchestration of a real-time workflow, including:

  • Data ingestion from the USGS Earthquake API.
  • Data transformation & modeling with dbt using the Medallion(Bronze–Silver–Gold) approach.
  • Pipeline orchestration with Apache Airflow for scheduling and monitoring.
  • Data storage & querying in PostgreSQL for structured analytics.
  • Interactive dashboards in Apache Superset (geospatial and statistical insights).
  • Scalable design extendable for streaming and large datasets.

The project showcases expertise in end-to-end data engineering, from raw ingestion to insightful analytics, with a focus on automation, reliability, and scalability.
